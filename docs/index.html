Can you speak like a violin?

Using a voice conversion method to perform timbre translation on musical sounds.

Here you will find some results produced by a timbre transfert project. The presented samples on this page are presented on datasets of 8 samples with their audio and respectives spectograms

Inputs

Used input dataset: 8 audio samples of voice and 8 audio samples of violin

Using the presented datasets, the model was treined with 3 different loss functions:

            
  • there is only the VAE component on the loss: L = lambda1 x L_VAE
    •         
  • there are the VAE and CC component: L = lambda1 x L_VAE + lambda1 x L_CC
    •         
  • complete model, with VAE and GAN: L = lambda1 x L_VAE + lambda1 x L_CC + lambda2 x L_GAN
    •     

For each loss function there were produced audios from 4 configurations

            
  1. original audio from voice database (encoder) and create a voiced audio (decoder)
    2.         
  2. original audio from voice database (encoder) and create a violin audio (decoder)
    3.         
  3. original audio from violin database (encoder) and create a voiced audio (decoder)
    4.         
  4. original audio from violin database (encoder) and create a violin audio (decoder)
